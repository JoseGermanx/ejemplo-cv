01. Acerca de mí

¡Hola! Soy Alex, un desarrollador de software con sede en Santiago. Mi interés por el desarrollo web comenzó allá por 2018 cuando intenté crear un blog personalizado para mi banda; resulta que modificar temas de WordPress me enseñó mucho sobre HTML y CSS.

Hoy en día, he tenido el privilegio de trabajar en una agencia de publicidad, una start-up y una gran corporación. Mi enfoque principal en este momento es construir productos accesibles e inclusivos y experiencias digitales para una variedad de clientes.

Habilidades Técnicas

JavaScript (ES6+) TypeScript React.js Node.js Python Tailwind CSS PostgreSQL Git & GitHub
